# WNS Analytics

### Problem Statement
 
 Predict the employees to be promoted.
 
### Dataset
 - train.csv : 54,808 records with paramters with whether the employee is promoted or not.
 - test.csv: 23,490 records to predict whether the person would be promoted.
### Results

Evalution was done with AUC Score.
AUC score: 0.509
